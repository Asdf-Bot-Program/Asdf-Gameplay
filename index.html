H.E.L.M.
This is a hack-and-slash RPG created in 2021 as part of a class project. This game includes multiple characters with different abilities and parkour elements such as climbing up walls and larger enemies. Each character has their own separate selection of abilities to fight such as melee or ranged abilities. There is also a dedicated money system and shop that can then be used to buy items. I am the sole creator of all of the programming, model implementation, animation blueprints, and UI design. This project was shown off at the 2022 East Coast Game Conference and at Old Dominion University 2022 Game Expo.

Point Noctis
This was a puzzle platformer created for ODU's Mars Fest 2022 Game Jam in only a week. It includes multiple time and gravity manipulation abilities such as gravity reduction and slow down time. Da-Bloons
Da-Bloons is a recent game inspired by the TikTok and Instagram trend of using a fake system of currency, known as Dabloons, while users are on the app. This project was created in the Unreal Engine 4 and plans to be expanded upon to include user created encounters as well as different levels. The project attempts to turn that experience into an actual game experience using a randomized algorithm similar to TikTok to determine the player's experience with the game. On top of this randomized algorithm the game also features a full inventory system that allows players to use the Dabloons they earn to buy items that they can then use to progress further. This project was shown off at the 2023 East Coast Game Conference.

Zephyr Run
This is a survival first person shooter created as the final project for my THEA 395 class. The game was created in Unreal Engine 4 and implements multiple complex models and animations. The game focuses on your character using an automatic assault rifle to survive against hordes of robots that are designed to kill. The game features different movement options such as a crouch and sprint option as well as a full animation state machine for both the player character and the enemy AI. It also features a full conservative ammo system that keeps an accurate track of the players ammo as well as a regenerating shield system.

Justice Well Played
This is a game project where I have been working with the nonprofit criminal justice organization, Impartial, owned by Diane Wells. Monarch Odyssey
This is a project for the Center for High Impact Practices and Old Dominion University designed by Dr. Moberly and the students of the Game Studies and Design major. The game is an Augmented Reality game with the setting being the ODU campus that allows for students to be better informed of the buildings and facilities around the campus. The game also features a multitude of mini games such as a creature befriending mini game and a stacking mini game. This game is programmed in the Unity Engine and uses the Mapbox system for navigation and mapping. For this project I was the Lead Programmer for the Spring 2022 semester and led the programming team to create the initial concepts for the Building Capture Mechanic, the Creature Befriending Mini-Game, and the Stacking Mini-Game. As well as working with the other teams to implement UI assets, 3D and 2D Models, and Music.
